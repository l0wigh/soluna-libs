# Math Library

Simple math library to help get more from simple arithmetics in Soluna

## Functions

- `(round Number)`
- `(ceil Number)`
- `(floor Number)`
- `(pow Number Number)`
- `(sqrt Number)`
- `(abs Number)`
- `(neg Number)`
- `(clamp Number)`
- `(max Number, Number)`
- `(min Number, Number)`
