# Official Soluna Libraries

This repo will contain every official libraries that can be imported in your projects

You will be able to use `soluna add libname` to automatically import the library you need

This will only work in Soluna project folders.

## Libraries list

- [math](https://github.com/l0wigh/soluna-libs/tree/master/libs/math)
- [random](https://github.com/l0wigh/soluna-libs/tree/master/libs/random)
- [json](https://github.com/l0wigh/soluna-libs/tree/master/libs/json)
